# Promotion Package Template

## Sections

- Company snapshot
- Source path and discovery context
- Thesis fit
- Screening score and rationale
- Novelty and dedupe result
- Relationship context
- Evidence links
- Known risks and unknowns
- Recommended Deal Room owner
- Suggested next task

## Approval Rule

Promotion should create or update a downstream opportunity only after a human approves the package and confirms the intended Deal Room boundary.
