# Source Registry Template

## Columns

- Source key
- Source type
- Owner
- Connection or credential state
- Scope and filters
- Cadence
- Budget or usage limit
- Review policy
- Last successful run
- Known issues
- Enablement status

## Usage

Keep this registry compact and operational. Detailed source-specific configuration can live in linked setup notes or task artifacts.
