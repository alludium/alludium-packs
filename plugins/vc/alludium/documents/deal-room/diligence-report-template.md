# Diligence Report Template

## Sections

- Workstream scope
- Sources reviewed
- Key findings
- Disconfirming evidence
- Open questions
- Risk severity
- Required follow-up
- Impact on investment recommendation

## Workstream Variants

Use the same structure for commercial, technical, financial, founder, legal, and market diligence. Add specialist sections only where the evidence requires it.
