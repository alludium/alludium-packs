# Financial Evaluation Guide

## Purpose

Use this guide to evaluate financial plausibility before formal diligence. The output should identify whether the financing plan, valuation, runway, and forecast assumptions can support the next investment decision.

## Core Questions

| Area | Evaluation Question | Evidence To Seek |
| --- | --- | --- |
| Current financial state | What is known about revenue, cash, burn, runway, and financing history? | Deck, model summary, bank/cash statement if available, financing notes. |
| Forecast realism | Are revenue, margin, hiring, and expense assumptions plausible? | Forecast bridge, customer-count assumptions, pricing, hiring plan, benchmark checks. |
| Use of funds | Does the raise fund the milestones needed for the next round? | Use-of-funds allocation, milestone plan, runway estimate. |
| Valuation and ownership | Does the valuation allow fund ownership and return potential? | Round size, cap table, valuation, option pool, ownership target, exit sensitivity. |
| Financing risk | What future capital is needed and when? | Forecast, runway, next-round assumptions, syndicate context. |
| Unit economics | Does the business model suggest attractive margins and repeatability? | Pricing, gross margin estimate, implementation cost, payback assumptions. |

## Output Standard

| Section | Required Content |
| --- | --- |
| Current financial view | Positive / mixed / negative / unknown. |
| Main financial thesis | The financial belief that must be true. |
| Evidence and confidence | Source-backed evidence with High / Medium / Low confidence. |
| Assumption risks | Unsupported forecast, pricing, hiring, expense, or financing assumptions. |
| Next proof needed | Model, cap table, cash evidence, forecast assumptions, customer pipeline, or sensitivity case. |
| Decision impact | Whether the opportunity can proceed, needs financial clarification, or should pause. |

## Boundary

Financial evaluation is not financial diligence. Do not verify historicals, bank statements, cap table accuracy, tax matters, or legal financing rights unless those source artifacts were reviewed.
