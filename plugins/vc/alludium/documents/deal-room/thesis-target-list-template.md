# Thesis Target List Template

## Sections

- Thesis and scope
- Search method
- Target company list
- Fit rationale
- Source links
- Warm intro paths
- Confidence notes
- Exclusions and rejected candidates
- Recommended next actions

## Standard

Treat the target list as a reviewable sourcing artifact. Include enough evidence for a human to decide whether each target deserves outreach or deeper research.
