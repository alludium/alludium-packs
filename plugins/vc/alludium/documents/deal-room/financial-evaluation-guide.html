Financial Evaluation Guide

Lightweight financial evaluation guidance for forecast plausibility, burn, runway, use of funds, valuation, ownership, and financing risk.

When this source is used as a rendered task output, create the final deliverable as a standalone safe static HTML artifact with a .html filename and mimeType: "text/html". Use the shared VC document styling rules from Template Use Guidance and make any generated links open in a new tab or window with target="_blank" and rel="noopener noreferrer". Do not create Markdown output artifacts from this source.

Purpose

Use this guide to evaluate financial plausibility before formal diligence. The output should identify whether the financing plan, valuation, runway, and forecast assumptions can support the next investment decision.

Core Questions

AreaEvaluation QuestionEvidence To Seek
Current financial stateWhat is known about revenue, cash, burn, runway, and financing history?Deck, model summary, bank/cash statement if available, financing notes.
Forecast realismAre revenue, margin, hiring, and expense assumptions plausible?Forecast bridge, customer-count assumptions, pricing, hiring plan, benchmark checks.
Use of fundsDoes the raise fund the milestones needed for the next round?Use-of-funds allocation, milestone plan, runway estimate.
Valuation and ownershipDoes the valuation allow fund ownership and return potential?Round size, cap table, valuation, option pool, ownership target, exit sensitivity.
Financing riskWhat future capital is needed and when?Forecast, runway, next-round assumptions, syndicate context.
Unit economicsDoes the business model suggest attractive margins and repeatability?Pricing, gross margin estimate, implementation cost, payback assumptions.

Output Standard

SectionRequired Content
Current financial viewPositive / mixed / negative / unknown.
Main financial thesisThe financial belief that must be true.
Evidence and confidenceSource-backed evidence with High / Medium / Low confidence.
Assumption risksUnsupported forecast, pricing, hiring, expense, or financing assumptions.
Next proof neededModel, cap table, cash evidence, forecast assumptions, customer pipeline, or sensitivity case.
Decision impactWhether the opportunity can proceed, needs financial clarification, or should pause.

Boundary

Financial evaluation is not financial diligence. Do not verify historicals, bank statements, cap table accuracy, tax matters, or legal financing rights unless those source artifacts were reviewed.
