Evidence And Citation Style Guide

Style guide for evidence, citations, confidence, and claim hygiene.

When this source is used as a rendered task output, create the final deliverable as a standalone safe static HTML artifact with a .html filename and mimeType: "text/html". Use the shared VC document styling rules from Template Use Guidance and make any generated links open in a new tab or window with target="_blank" and rel="noopener noreferrer". Do not create Markdown output artifacts from this source.

Purpose

Use this guide whenever a VC task produces a memo, scorecard, digest, diligence report, decision record, promotion package, or review artifact. The document should make it easy for an investment team to distinguish evidence, inference, assumption, stale information, and open risk.

Citation Rules

Claim TypeRequired Treatment
Numeric claimCite source, period, units, and artifact or URL. Recompute derived values if used in a recommendation.
Founder or management claimLabel as founder-provided unless independently verified.
Third-party evidenceCite source name, date, and link or artifact ID. Note source quality if weak.
InferenceUse "evidence suggests" or "we infer" and name the evidence base.
AssumptionLabel as assumption and state what would confirm or falsify it.
Stale sourceKeep visible with date and explain whether it is still usable.

Source Precedence For Company-Claimed Facts

Company-claimed facts are facts a company asserts about itself: ARR and other traction metrics, fundraising ask, round and stage, customer counts and named customers, roadmap, go-to-market, founder names and roles, and team narrative.

For these facts, uploaded and company-provided material is the primary source:

  1. Founder/company-provided materials (decks, data rooms, founder docs, call notes)
  2. Firm CRM/deal-system notes and records
  3. First-party company website
  4. High-quality third-party sources
  5. Broad web search

Public, structured, and web sources corroborate, challenge, timestamp, or fill gaps in company-claimed facts. They must not silently replace a company-provided value. When a public or structured source conflicts with company-provided material on a company-claimed fact, keep the company-provided value as the primary claim, show the conflicting value and its source, and explain which value the artifact uses and why. Escalate material conflicts for human review rather than overwriting them.

This precedence applies only to company-claimed facts. For independent validation facts — press, market context, incorporation/registry data, third-party headcount signals — high-quality external sources may outrank or challenge company materials.

Preferred Language

Use concise investment language. Prefer "evidence suggests" over "proves" unless the source is definitive. Use "not yet validated" for gaps. Use "requires partner judgment" when a decision depends on fund strategy rather than task evidence.
