# Evidence And Citation Style Guide

## Purpose

Use this guide whenever a VC task produces a memo, scorecard, digest, diligence report, decision record, or promotion package. The document should make it easy for an investment team to distinguish evidence, inference, assumption, and open risk.

## Citation Rules

- Cite material claims with a source name, date when available, and artifact or URL reference.
- Mark unsupported claims as assumptions or hypotheses.
- Separate founder-provided claims from third-party evidence.
- Use confidence labels only when the basis for confidence is visible.
- Keep stale evidence visible instead of silently reusing it.

## Preferred Language

Use concise investment language. Prefer "evidence suggests" over "proves" unless the source is definitive. Use "not yet validated" for gaps. Use "requires partner judgment" when a decision depends on fund strategy rather than task evidence.
